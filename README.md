# OldSpace

Ретро веб-интерфейс в стиле Windows 95/98 на чистом JavaScript.

## Описание

OldSpace — это легковесный веб-фреймворк, воссоздающий классический интерфейс Windows 95/98 с использованием современных веб-технологий.

### Возможности

- **UIObject** — базовый класс для всех UI-элементов
- **Form** — окна с заголовком, перетаскивание и изменение размера
- **Button** — кнопки в ретро-стиле с 3D-эффектом
- **TextBox** — текстовые поля с объёмными границами
- Темизация через `client_config.json`
- CSP-совместимая архитектура
- Модульная структура (ядро + приложения)

## Структура проекта

```
drive_root/        # Ядро движка
drive_forms/       # Демо-приложение с UI-компонентами
  ├── index.html
  ├── client.js
  ├── UI_classes.js
  ├── UI.js
  ├── style.css
  ├── client_config.json
  └── server_config.json
main_server.js     # Главный сервер
server.config.json # Конфигурация
```

## Запуск

```bash
node main_server.js
```

Откройте браузер и перейдите по адресу `http://localhost:8080`

## Технологии

- Node.js (HTTP-сервер)
- Vanilla JavaScript
- Canvas API (для иконок кнопок)
- CSS (ретро-стилизация)
